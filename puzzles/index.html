Chess Woodpecker App

This tool helps you improve your pattern recognition using the Woodpecker Method.
By repeatedly solving the same set of puzzles, you’ll become faster at recognizing key tactical motifs like forks, pins, skewers, and discovered attacks!

Note: This app is still in early development, and many improvements are being worked on. Please be aware that currently, this app does not function well on mobile devices. Specifically, the Lichess app may launch on mobile, instead of opening puzzles in a new tab. A mobile-compatible version will be available soon. Additionally, currently your session is only preserved for as long as you remain on this page.

How to Use
  1. Click the Start Session button to begin.
  2. A new tab will open with a Lichess puzzle.
  3. After solving, return here and mark it as
    Success if correct, or
    Fail if not.
  4. The next puzzle will load automatically once you mark the current one.
  5. Click the Stop Session button to end the session.
